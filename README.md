# SGPP - Sistema de Gestión de Prácticas Profesionales

## Descripción
Sistema web desarrollado en Flask para gestionar estudiantes, empresas y prácticas.

## Tecnologías
- Flask
- MySQL (XAMPP)
- SQLAlchemy
- HTML, CSS

## Instalación
1. pip install -r requirements.txt
2. Crear BD sgpp en XAMPP
3. Importar sgpp.sql
4. python run.py
