# SGPP - Sistema de Gestión de Prácticas Profesionales

## Descripción
Sistema web desarrollado en Flask para la gestión de estudiantes, empresas y prácticas preprofesionales, con autenticación mediante JWT.
## Tecnologías
- Flask
- PyMySQL / MySQL (XAMPP)
- Flask-JWT-Extended
- HTML, CSS, JavaScript
- Ninja 2
